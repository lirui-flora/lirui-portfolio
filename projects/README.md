# Li Rui Portfolio

这是 Li Rui 的个人作品集，包含两个主要项目：

1. **Movie 小推荐系统**
   - 数据采集、文本处理、聚类分析、情感分析
   - 基于聚类与多特征实现了一个简单的推荐系统
   - 详细内容请参见 [movie/README.md](movie/README.md)

2. **光伏发电优化（pv_opt）**
   - 使用 MATLAB 进行光伏预测、优化与情景分析
   - 涵盖时间序列预测（指数平滑、LSTM）、优化模型、敏感性分析
   - 详细内容请参见 [pv_opt/README.md](pv_opt/README.md)

## 技术栈

- Python: pandas, numpy, scikit-learn, matplotlib, seaborn, jieba, wordcloud, transformers, torch
- MATLAB: Deep Learning Toolbox, Time Series Toolbox
- Git & GitHub: 版本管理与远程仓库管理

## 项目运行方式

- **Movie 推荐系统**：进入 `movie/`，按照 `README.md` 指示创建环境并运行脚本。
- **光伏优化**：进入 `pv_opt/`，在 MATLAB 中运行对应脚本。
